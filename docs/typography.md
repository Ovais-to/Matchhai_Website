# Typography

## Typefaces
- Heading (type for a moment): Montserrat, weights 600 and 700
- Body (type to live with): Raleway, weights 400, 600, 700
- Fallbacks: `"Segoe UI", sans-serif`

## Usage
- Headings: use semantic `h1`-`h6` and rely on global heading styles
- Body copy: default `body` styles are the baseline
- Long-form content: wrap with `.reading` for measure and rhythm

## Utility Classes
- `.section-title`: primary section heading
- `.page-title`: page heading (legal pages)
- `.section-lede`: short intro paragraph under headings
- `.section-kicker`: small, uppercase eyebrow
- `.reading`: long-form blocks with 65ch measure and 1.7 line-height

## Checklist
- Body size at or above 16px, prefer 17-18px in `.reading`
- Line height at or above 1.6 for body, 1.7 for long-form
- Line length between 45-75 characters (`max-width: 65ch`)
- Clear hierarchy between heading levels
- Links remain visually distinct
- Font payload limited to used weights only
